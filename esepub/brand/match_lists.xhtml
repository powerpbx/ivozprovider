Listas coincidencia genéricas

Las Listas de coincidencia estan diseñadas para agrupar números conocidos o patrones numéricos para asociarlos a distintos tratamientos.

Los administradores de marca pueden crear listas de coincidencia genéricas para compartir por todas sus compañías. Esto puede resultar especialmente útil para números de listas negras o configuraciones globales de Listas de coincidencia para ACLs.
